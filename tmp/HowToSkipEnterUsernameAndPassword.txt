Step1. Set HOME variable in your command line.
	setx HOME %USERPROFILE%

Step2. Create netrc file in the HOME path that we setup in Step 1.
    The netrc file is _netrc for windows; .netrc for linux

Step3. Modify the netrc file. Please keyin GIT remote repository account and password.
    machine <servier name>
    login <username>
    password <password>

i.g.
machine gitlab.kosmossystems.com
login hyang
